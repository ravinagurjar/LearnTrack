About LearnTrack

LearnTrack is a web-based learning tracker developed using Python Flask, SQLite, HTML and CSS. It helps students record their learning activities, monitor their progress and maintain consistency in their studies.


Project Objectives

  • Record daily learning activities.
  • Track learning progress.
  • Maintain a personal learning journal.
  • Search and manage learning entries.
  • Improve consistency through self-reflection.

Key Features

  • Add new learning entries.
  • Edit existing entries.
  • Delete learning entries.
  • Search entries by skill or category.
  • View learning progress.
  • Simple and responsive user interface.

Technology Stack

Technology Purpose
Python Backend Programming
Flask Web Framework
SQLite Database
HTML Web Page Structure
CSS User Interface Design

Developed By

This project has been developed as a second-year B.Tech Computer Science mini project to demonstrate CRUD operations using Flask with SQLite.
